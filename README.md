Вариант 1. Использовать screen (если установлен)
Запусти экран сессией (например, dance_bot):

screen -S dance_bot

Внутри сессии запусти бота:

python3 /home/baikhonov/dance_bot.py

Чтобы отсоединиться от экрана (бот продолжит работать в фоне):

Нажми Ctrl + A, потом D.

Чтобы вернуться к сессии позже:

screen -r dance_bot
