Telegram bot для студии танцев BachataMania

## Локальное тестирование

1. Откройте терминал в папке проекта:
2. Убедитесь, что в файле `.env` указан токен:
   - `TELEGRAM_TOKEN=...`
3. При необходимости установите зависимости:
   - `pip install python-telegram-bot python-dotenv`
4. Запустите бота:
   - `python dance_bot.py`
5. Проверьте работу в Telegram:
   - откройте вашего бота;
   - отправьте `/start`;
   - проверьте кнопки (`📍 Адрес`, `💸 Цены`, `📅 Расписание` и другие).

### Полезно для отладки

- Логи выводятся в консоль (включен уровень `INFO`).
- Остановка бота: `Ctrl + C`.
